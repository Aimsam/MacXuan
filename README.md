# Mac 网盘下载客户端
[简介和打包](http://soft.idealweb.cn/)
这是一款用js写的mac网盘下载工具，一个小玩具，没有精力维护下去了。>_<!。
所有用到的外部程序都打包进去了，运行方法:/path/nw.js  .


## 工具
* 开源的下载工具[aria2](https://github.com/tatsuhiro-t/aria2)
* 界面[NW.js](https://github.com/nwjs/nw.js)
* angular.js
